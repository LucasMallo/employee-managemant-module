
# Employee managemant module

## Installation
Run:
```
composer install
```
and
```
npm install
```

## Database
After setting database in .env run migrations:
```
php artisan migrate
```
and seeders (for demo data):
```
php artisan db:seed
```

## Run
For running application, start:
```
php artisan serve
```
and
```
npm run dev
```
